RAG Research Architecture

Pipeline flowcharts, color-coded by block

The same color always means the same architectural responsibility: Document is amber, Retrieval is teal, Answering is coral, Reasoning & orchestration is blue, and branching or control logic is slate. That makes the differences between P0 and P5_ver1 readable at a glance.

Color Legend

Document block PDF extraction, chunking, metadata, and Chroma build.
Retrieval block Query embedding, dense search, sparse search, fusion, and selection.
Answering block Context assembly, draft generation, answer formatting, and evidence output.
Reasoning & orchestration LangGraph routing, next-step choice, and loop structure.
Decision logic Conditional branches such as critique gating or loop outcomes.

Shared Block View

All versions share this basic structure. The later versions do not replace the four blocks, they mainly make retrieval smarter and orchestration more agentic.

Architecture skeleton
